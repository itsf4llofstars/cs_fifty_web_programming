The language sass is an extension of CSS

Use the filename filename.scss

This new *.scss file will need compiling

Install the compiler sass program and at the prompt enter:

$ sass file.scss:file.css

or for auto-compile on changes

$ sass --watch file.scss:file.css

After compilation, you will have to change the link file name to file.css
